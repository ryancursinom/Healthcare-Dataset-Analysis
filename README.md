# 📊 Healthcare Dataset Analysis

Projeto de análise de dados voltado para extração de insights a partir de um dataset da área da saúde utilizando Python e Jupyter Notebook.

O projeto busca analisar informações de pacientes, hospitais, condições médicas, custos hospitalares e indicadores operacionais através de técnicas de análise de dados e visualização.



## Objetivo

O objetivo deste projeto é analisar dados da área da saúde para:

- Identificar padrões em atendimentos médicos
- Avaliar indicadores operacionais hospitalares
- Gerar insights a partir dos dados
- Aplicar estatística descritiva em um cenário real



## Dataset

O dataset contém registros relacionados a pacientes, hospitais, condições médicas, seguros de saúde, custos e tratamentos.

### Principais atributos do dataset

| Coluna | Descrição |
|---|---|
| Age | Idade |
| Gender | Gênero |
| Chronic Condition | Condição médica |
| Total Cost | Valor cobrado |
| Admission Type | Tipo de admissão |
| Medication Count | Quantidade de Medicação |
| Satisfaction Score | Nivel de Satisfação |


## Análises Realizadas

O projeto inclui análises como:

- Limpeza e tratamento de dados
- Estatísticas descritivas
- Média de tempo de espera
- Desvio padrão
- Análise por departamentos
- Frequência de condições médicas
- Análise de custos hospitalares
- Distribuição demográfica dos pacientes
- Visualização de tendências hospitalares


## KPIs Analisados

### - Tempo médio de espera
Avaliação do tempo médio de atendimento dos pacientes.

### - Desvio padrão do tempo de espera
Medição da variabilidade no tempo de espera.

###  - Performance por departamento
Comparação entre departamentos hospitalares.

### - Análise de faturamento
Avaliação dos custos médicos e cobertura dos planos de saúde.

### - Perfil dos pacientes
Análise de idade, gênero e distribuição demográfica.

## Principais Insights

Alguns insights que podem ser obtidos durante a análise:

- Alguns departamentos apresentam maior tempo médio de espera
- Certas condições médicas possuem maior incidência
- Os custos hospitalares variam significativamente conforme o tipo de admissão
- Existem diferenças de desempenho entre hospitais
- Faixas etárias específicas apresentam padrões distintos de atendimento
